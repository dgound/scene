./SCENE is an attempt at archiving the Greek underground music scene using a data-driven approach.

Method

Graph

The scene is modelled as a graph in which each node represents a band. For the moment, only bands that have released at least one EP are considered.

Nodes & edges

An edge connects two nodes whenever they share at least one member — whether a permanent member or a guest on an official release.

Weights

Edges are weighted by the number of members shared between two bands. Thicker edges mean more shared members. Nodes are sized by their number of neighbours, so larger nodes belong to bands with more collaborations and side projects.

Tools

The graph was created with the pyvis Python library and post-produced with HTML and CSS.

Contribute

./SCENE is a work in progress. If you would like to submit info for additional bands or send feedback or corrections, write to enable JavaScript for email.
