./SCENE is an attempt at archiving the Greek underground music scene using a data-driven approach.

Method

Graph

The scene is modelled as a graph where each node represents a band. For the moment, only bands that have at least one EP released are considered.

Nodes and edges

An edge is placed between two nodes if they share at least one band member (either as a full-time member or as a guest on an official release).

Weights

Edges are weighted by the number of members shared between two bands — thicker edges correspond to more shared members. Nodes are sized by their number of neighbours, so larger nodes correspond to bands with more collaborations and side projects.

Tools

The graph was created using the pyvis Python library. The output was post-produced with HTML5 and CSS3.

./SCENE is a work in progress. If you would like to submit info for additional bands or provide feedback/corrections, please write to: enable JavaScript for email.
